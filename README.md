# WriteFlow - SaaS 랜딩 페이지

AI 기반 글쓰기 어시스턴트인 "WriteFlow"를 위한 모던하고 반응형이며 웹 접근성이 뛰어난 랜딩 페이지입니다. **Next.js**, **Tailwind CSS v4**, **Framer Motion**으로 제작되었습니다.

## 🚀 기능 (Features)

- **아름다운 히어로 섹션**: 부드러운 등장 애니메이션과 커스텀 CSS/HTML 목업(Mock UI) 그래픽.
- **기능(Features) 그리드**: Lucide React 아이콘을 활용한 시간차(staggered) 스프링 애니메이션.
- **인터랙티브 요금제(Pricing)**: 월간/연간 요금제를 전환할 수 있는 부드러운 동작의 토글 스위치와 강조된 인기(Popular) 플랜 기능.
- **FAQ 아코디언**: 매끄러운 펼치기 및 접기 애니메이션을 위해 `framer-motion`의 `AnimatePresence`를 사용하여 구축.
- **완벽한 반응형**: Tailwind CSS 유틸리티 클래스를 사용하여 모바일, 태블릿, 데스크톱 환경에 맞게 최적화됨.

## 🛠 기술 스택

- **프레임워크**: [Next.js App Router](https://nextjs.org/) (React 19)
- **스타일링**: [Tailwind CSS v4](https://tailwindcss.com/)
- **애니메이션**: [Framer Motion](https://www.framer.com/motion/)
- **아이콘**: [Lucide React](https://lucide.dev/)

## 🏃‍♂️ 시작하기

1. **의존성 패키지 설치:**
   ```bash
   npm install
   ```
2. **개발 서버 실행:**
   ```bash
   npm run dev
   ```
3. 브라우저에서 [http://localhost:3000](http://localhost:3000) 주소로 접속하여 결과를 확인하세요.
