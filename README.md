# WriteFlow - SaaS Landing Page

A modern, responsive, and highly interactive landing page for "WriteFlow," an AI-powered writing assistant. Built with **Next.js**, **Tailwind CSS v4**, and **Framer Motion**.

## 🚀 Features

- **Beautiful Hero Section**: Smooth entrance animations and a custom CSS/HTML mock UI graphic.
- **Features Grid**: Staggered spring animations utilizing Lucide React icons.
- **Interactive Pricing**: A fully functional toggle switch for Monthly/Yearly pricing with a highlighted Popular plan.
- **FAQ Accordion**: Built with `framer-motion`'s `AnimatePresence` for seamless expand and collapse animations.
- **Fully Responsive**: Optimized for Mobile, Tablet, and Desktop using Tailwind CSS utility classes.

## 🛠 Tech Stack

- **Framework**: [Next.js App Router](https://nextjs.org/) (React 19)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🏃‍♂️ Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Run the development server:**
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
